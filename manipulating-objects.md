## Accessing Object Properties and Methods
### Properties
`obj.property` or `obj[property]`

### Methods
`obj.method()`

## Adding Object Properties
`obj.newProperty = value;`

`obj.newMethod = function() { // do something };`

`obj['newProperty'] = value;`

`obj[variable] = value;`


## Delete Obejct Properties and Methods
`delete obj.property`

`delete obj['property']`
